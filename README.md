# Testing, Linux and Servers Assignment

## Overview
This repository contains the implementation of system monitoring, user management, and automated backup configuration as part of the DevOps Linux assignment.

## Task 1: System Monitoring
- Installed and configured monitoring tools such as htop and nmon
- Monitored CPU, memory, disk usage using df and du
- Logged system metrics for capacity planning

## Task 2: User Management and Access Control
- Created users Sarah and Mike
- Configured isolated workspaces with restricted permissions
- Enforced password expiry policy

## Task 3: Backup Configuration
- Automated Apache and Nginx backups using shell scripts
- Scheduled weekly backups using cron
- Verified backup integrity using tar
